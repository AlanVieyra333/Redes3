\begin{UseCase}{GV-CU1}{Gestionar vuelo}{
		Permite al \refElem{Cliente} gestionar solicitudes, cancelar la
		solicitud actual (o que se lleva a cabo en un instante de 
		tiempo), monitorear o consultar una solicitud que se lleva a cabo
		en un momento determinado. Para gestionar solicitudes en necesario
		establecer una ubicación destino, ya que permitirá saber al 
		\refElem{Drone} a cuál lugar llegar.
