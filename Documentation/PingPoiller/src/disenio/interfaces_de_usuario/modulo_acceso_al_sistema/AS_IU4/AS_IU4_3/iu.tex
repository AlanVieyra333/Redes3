\subsection{AS-IU4.3 Actualizar contraseña}

\subsubsection{Objetivo}
Esta pantalla permite al \refElem{Cliente} actualizar su contraseña.

\subsubsection{Diseño}
En la figura \ref{AS-IU4.3} se muestra la pantalla 'Actualizar contraseña', en la cual el \refElem{Cliente} introduce su nueva contraseña.


\IUfig[.8]{gui/IU4/IU4_3_Actualizar_contrasenia}{AS-IU4.3}
{Actualizar contraseña.} 

\subsubsection{Opciones}
\begin{itemize}
	\item \IUbutton{Actualizar contraseña}: Actualiza la contraseña del cliente en el sistema.
\end{itemize}
