\subsection{AS-IU4.2 Verificar correo}

\subsubsection{Objetivo}
Esta pantalla permite al \refElem{Cliente} verificar su cuenta para permitirle restablecer su contraseña.

\subsubsection{Diseño}
En la figura \ref{AS-IU4.2} se muestra la pantalla "Verificar correo", en la cual el \refElem{Cliente} introducir su correo para que el sistema lo vefifique y permita restablecer su contraseña.


\IUfig[.8]{gui/IU4/IU4_2_Verificar_correo}{AS-IU4.2}
{Verificar correo.} 

\subsubsection{Opciones}
\begin{itemize}
	\item \IUbutton{Siguiente}: Conrinuar con el siguiente paso que es la restauración de la contraseña.
\end{itemize}
