Se consideran los siguientes dispositivos de hardware para el desarrollo de 
prototipos físicos:

	\item Sensor ultrasónico HC-SR04.
	\item Interfaz USB de comunicación entre el drone y el microcontrolador.
	\item Módulo GSM-800L.
\end{itemize}
